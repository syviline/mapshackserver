/shop
get - нужен для получения информации о магазине.
Входные: id магазина
Выходные: вся инфа о магазине, примерно: {
	id: string,
	name: string,
	description: string,
	image(нужно подумать в каком виде),
	marks: [{
		name: string
		description: string,
		position: {
			lat: float(дробное),
			lng:  float(дробное)
		}
	}]
}

del - удаляет магазин по id.
Входные: id: string
Выходные: status: string

put - Добавить метку магазина.
Входные: {
	id: string,
	name: string,
	description: string,
	website: string,	
	image(нужно подумать в каком виде),
	marks: [{
		name: string
		description: string,
		position: {
			lat: float(дробное),
			lng:  float(дробное)
		}
	}],
	position: {
		lat: float(дробное),
		lng:  float(дробное)
	}
}
Выходные: status
